# 🛒 Stationery E-Commerce Website

A responsive stationery e-commerce website developed to simulate an online shopping experience. The project includes product browsing, category filtering, shopping cart management, and a checkout process while focusing on responsive design and user experience.

## 🚀 Features

- Product catalog browsing
- Product search functionality
- Category filtering
- Dynamic product detail pages
- Add-to-cart functionality
- Cart management
- Checkout workflow
- Responsive design
